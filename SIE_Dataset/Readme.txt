#Learning to remove sandstorm for image enhancement
#https://doi.org/10.1007/s00371-022-02448-8
#(Online publication at The Visual Computer,2022)

## Datasets
Sandstorm Image Enhancement (SIE) dataset which covers 460 real-world
images and 500 synthetic images with references.

1.Real_world_images (460)
(1)Floating_dust-150
(2)Blowing_sand-150
(3)Sandstorm-150
(4)Strong_sandstorm-10

2.Synthetic_images (500)
(1)Ground_truth
(2)Sand_dust_images

## Highlights
In this paper, the images were first resized to experiment. For fairness, the  SIE dataset is resized as input in other contrasting methods. 

Run:"resize.py" to resize images. If you do not have the above requirements, you can ignore it.
